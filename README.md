# STARSEC

- **!! Starsec is currently in development, check back to see the first releae after november 2026 !!**

STARSEC is a live cybersecurity competition platform where teams operate massive starships (e.g., Imperial Star Destroyer) from within the ship's bridge. It combines real Docker-based cyber attacks with a fully immersive 3D simulation — designed to train cyber students, host at events like Rowdy Cybercon, and serve as a live AV production for streaming.

Overview
Two teams face off in space. Each team operates a capital starship with a full suite of containerized services (reactors, weapons, shields, comms, etc.). Operators perform real cyber attacks against the enemy team's Docker compose network via in-game terminals, while the 3D simulation visualizes every turbolaser bolt, shield impact, and fighter dogfight.

Cyber Layer: Real Docker containers, real pentesting tools (nmap, metasploit, sqlmap), real firewalls (iptables, snort IDS)
Game Layer: 3D starship models, particle-based weapons, shield shaders, physics-based fighters
AV Layer: Multi-camera cinematic mode, Spout GPU texture sharing → OBS Studio for live streaming
Audience Layer: Web browser-based fighter cockpit for audience participation
